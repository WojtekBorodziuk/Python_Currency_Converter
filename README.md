# Python_Currency_Converter
Currency Converter with API
Prosty konwerter walut napisany w Pythonie z wykorzystaniem biblioteki `tkinter` (interfejs graficzny) oraz `requests` (pobieranie danych z API).

Funkcjonalność:
- Pobiera aktualne kursy walut z API: [https://api.exchangerate-api.com/](https://api.exchangerate-api.com/)
- Przelicza kwoty między wybranymi walutami
- Wyświetla aktualne kursy najpopularniejszych walut
- Intuicyjny interfejs graficzny
